# SkillBridge Sri Lanka

A beginner-friendly multi-page frontend website built using **HTML**, **CSS**, and **JavaScript**.

---

## Overview

SkillBridge Sri Lanka is a concept website designed to help students explore career opportunities and required skills while practicing modern frontend web development.

This project was created as part of my frontend learning journey.

---

## Features

- 🏠 Landing Page
- 💼 Careers Page
- 📚 Skills Page
- 📩 Contact Page
- 🎨 Responsive Layout
- ⚡ Basic JavaScript Interactions
- 🧭 Multi-page Navigation

---

- CSS3
- JavaScript
